# Planned Premium Features

## Tagging system

**Responsibility when built:**

- **Organize notes and subjects** — Premium users can add tags (e.g. `#exam2`, `#chapter3`, `#formulas`) to notes or subjects.
- **Filter and find** — Filter the note list or dashboard by one or more tags (e.g. “show all notes tagged #midterm”).
- **Faster navigation** — Use tags instead of only subject/folder so the same note can sit in multiple “buckets” (e.g. a note tagged both #biology and #exam-review).
- **Optional:** Tag-based review sessions (e.g. “Review only #weak” or “Practice #formulas”) and search that includes tags.

So the tagging system is responsible for **labelling content**, **filtering by those labels**, and **faster organization** beyond the 3-subject / subject–notes structure.
